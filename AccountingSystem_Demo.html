🚀 Roadmap

Top 10 Improvements

Prioritized recommendations to make the system enterprise-ready.

1
🔐 Secure PIN Hashing (bcrypt)
User PINs are currently compared in plaintext. Implement bcrypt hashing — if the database is ever exposed, all user credentials would be compromised immediately.
CriticalEasy1-2 hours
2
🍪 Replace localStorage Auth with Secure Cookies
Session stored in localStorage is vulnerable to XSS attacks. Move to httpOnly, Secure cookies with CSRF token validation to prevent session hijacking.
CriticalMedium4-8 hours
3
📐 Balance Sheet Equation Validation
The system never verifies that Assets = Liabilities + Equity. Add automatic validation with a clear warning banner when this fundamental accounting equation is violated.
HighEasy2-3 hours
4
🏦 Opening Balances in Trial Balance
Trial balance currently only shows transaction totals, not opening balances. Add an "Opening Balance" column so the trial balance matches the complete ledger view from getAllAccountBalances.
HighEasy3-4 hours
5
📄 Pagination in Journal Entries List
All 496 entries load at once — will become slow with thousands of entries. Implement cursor-based pagination (50 entries/page) with "Load More" or page numbers for fast performance at scale.
HighMedium6-8 hours
6
🔄 Database Triggers for is_balanced
The is_balanced flag can drift if lines are modified outside the app. A Postgres trigger on journal_entry_lines would automatically keep the header in sync — eliminating the need for the manual "Repair" button.
MediumHard4-6 hours
7
💱 Multi-Currency Support
Add currency field to journal entries and accounts. Store amounts in base currency with exchange rates. Enable currency conversion in all financial reports with live exchange rate integration.
MediumHard2-3 weeks
8
🔒 Supabase Row Level Security (RLS)
Currently, any authenticated user can read all financial data via the anon key. Implement RLS policies so users only see data permitted by their role — essential for multi-company or multi-user deployments.
MediumHard1-2 days
9
📊 Budget vs. Actuals Module
Add a budget planning module where managers define monthly/annual budgets per account. The system automatically shows actual vs. budget variance in reports — critical for management decisions.
EnhancementHard1-2 weeks
10
🏦 Bank Reconciliation Module
Import bank statements (CSV/OFX) and automatically match them against cash account journal entries. Flag discrepancies, show unreconciled items, and generate reconciliation reports for auditors.
EnhancementHard2-3 weeks
